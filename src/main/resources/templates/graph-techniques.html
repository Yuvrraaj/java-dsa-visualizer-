Graph Implementation & Traversals

Graphs are usually implemented using adjacency lists and explored using BFS and DFS traversal algorithms.

1. Building an Adjacency List


public List<List<Integer>> buildGraph(int n, int[][] edges) {

    List<List<Integer>> adjList = new ArrayList<>();

    // Initialize empty lists for all nodes
    for (int i = 0; i < n; i++) {
        adjList.add(new ArrayList<>());
    }

    // Add edges (Undirected Graph)
    for (int[] edge : edges) {
        int u = edge[0];
        int v = edge[1];

        adjList.get(u).add(v);
        adjList.get(v).add(u);
    }

    return adjList;
}

2. Breadth-First Search (BFS)


public void bfs(List<List<Integer>> adjList, int startNode) {

    boolean[] visited = new boolean[adjList.size()];
    Queue<Integer> queue = new LinkedList<>();

    visited[startNode] = true;
    queue.add(startNode);

    while (!queue.isEmpty()) {
        int curr = queue.poll();
        System.out.println("Visited: " + curr);

        for (int neighbor : adjList.get(curr)) {
            if (!visited[neighbor]) {
                visited[neighbor] = true;
                queue.add(neighbor);
            }
        }
    }
}

3. Depth-First Search (DFS)


public void dfs(List<List<Integer>> adjList,
                int curr,
                boolean[] visited) {

    visited[curr] = true;
    System.out.println("Visited: " + curr);

    // Explore all neighbors recursively
    for (int neighbor : adjList.get(curr)) {
        if (!visited[neighbor]) {
            dfs(adjList, neighbor, visited);
        }
    }
}
