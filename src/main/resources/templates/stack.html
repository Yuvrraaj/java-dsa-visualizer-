A Stack is a linear data structure that follows a particular order in which the operations are performed. The order is LIFO (Last In First Out).

Internally, a Stack can be implemented using either Arrays (contiguous memory) or Linked Lists (dynamic memory allocation).

LIFO Principle

The last element added is the first one to be removed.

Restricted Access

You can only access, add, or remove the top element.

Time Complexity

Push, Pop, and Peek are all $\mathcal{O}(1)$ operations.
