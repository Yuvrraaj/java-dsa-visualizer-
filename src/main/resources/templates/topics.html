Data Structures

Select a data structure to explore its theory, interactive visualizer, techniques, and common problem patterns.

Arrays

Contiguous memory blocks. Great for quick access, slow for middle insertions.

ArrayLists

Dynamic arrays that resize automatically. The workhorse of Java collections.

Linked Lists

Nodes connected by pointers. Excellent for fast insertions and deletions.

Stacks

LIFO (Last In, First Out). Think undo buttons and execution calls.

Queues

FIFO (First In, First Out). Essential for scheduling and breadth-first search.

Trees

Hierarchical data. Enables rapid $\mathcal{O}(\log n)$ searching and sorting.

Graphs

Networks of interconnected nodes. Used for routing, social networks, and more.
